# plurk

A crystal library for Plurk OAuth API

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  plurk-oauth:
    github: akiicat/PlurkOAuth.cr
    branch: master
```

## Usage

```crystal
require "plurk-oauth"
```
